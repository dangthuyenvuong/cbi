# CBI-CLI

# Giới thiệu

- `CBI-cli` là bộ công cụ hỗ trợ quá trình code và deploy dự án Reactjs được phát triển bởi công ty ChiDoanh

# Cài đặt

> npm i cbi-cli -g


# Hướng dẫn sử dụng

> cbi -h
## Tạo dự án

> cbi new
## Tạo store (service, store)

> cbi g s
## Chuyển đổi giữa Reactjs thuần và Nextjs và ngược lại

> cbi convert
